## ğŸ› ï¸ Setup Instructions

### Step 1: Create Project Structure
1. Create a new folder named `portfolio` on your computer
2. Inside the `portfolio` folder, create these subfolders:
   - `css`
   - `js` (optional, JavaScript is inline)
   - `images`
   - `documents`

### Step 2: Add Files
1. **HTML**: Save the HTML code as `index.html` in the root `portfolio` folder
2. **CSS**: Save the CSS code as `styles.css` in the `css` folder
3. **Images**: Add your images to the `images` folder with these names:
   - `profile.jpg` - Your professional headshot
   - `project1.jpg` - First project image or video thumbnail
   - `project2.jpg` - Second project image or video thumbnail
   - `dancing.jpg` - Photo from dancing activities
   - `aiesec.jpg` - Photo from AIESEC activities
4. **Resume**: Add your resume PDF as `resume.pdf` in the `documents` folder

### Step 3: Customize Content
1. **Profile Image**: Replace `images/profile.jpg` with your professional photo
2. **Project Images**: Add relevant project images or video thumbnails
3. **Activity Photos**: Add photos from your extracurricular activities
4. **Resume**: Add your updated resume PDF
5. **Certificate Link**: Update the certificate link in the HTML (line with "View Certificate")

### Step 4: Test Locally
1. Open `index.html` in your web browser
2. Test all navigation links
3. Check responsiveness by resizing the browser window
4. Verify all images load correctly
5. Test the resume download link
